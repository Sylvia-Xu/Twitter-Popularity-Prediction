How to run our code:

1) Calculating statistics of hashtags: run statistics.py

2) Fit linear regression model with 5 given features: run linear_regression_given_features.py

3) First, run new_feature to write new features into documents. Then, run fit_model_new_feature to make prediction.

4) 10-fold cross validation: run cross_validation.py

5) Run model on test data: run test_data.py

6) Text Classification: run text_classification.py
